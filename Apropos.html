À propos - Qui suis-je ? Pourquoi j'ai fait ce site ?

Bonjour,

Je m'appelle Mounir Bouzid et je suis étudiant à Émica dans le programme de soutien informatique.

J'ai commencé ce programme car j'aime beaucoup l'informatique et je me vois bien travailler dans le domaine plus tard.

Dans le cadre du cours Moyens de télécommunication, j'ai fait ce site. Il s'adresse aux aux gens voulant s'informer sur l'utlité du langage sans l'apprendre. Voir ce qui peut être fait avec et se renseigner sur où commencer.

J'ai choisi le sujet Python car j'ai fait mon application dans un autre cours en utilisant le langage. J'ai beaucoup aimé mon expérience avec le langage et j'ai beaucoup appris.

J'ai donc décidé de vous montrer sur ce site les domaines dans lequel le langage Python est le plus utilisé ainsi que mon application.

Je suis conscient qu'en lisant le contenu du site, vous n'apprendrez pas à coder ou même toutes les branches dans lesquelles Python est utilisé, mais j'ai l'espoir que ça vous donnera envie de faire plus de recherches si vous voullez apprendre un langage de programmation.

Merci d'avoir consulté le site, bon apprentissage.



Ce site web est supporté par les navigateurs : Chrome, Opéra, Safari, Edge et FireFox.
