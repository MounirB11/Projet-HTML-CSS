Questions - F.A.Q

Qu'est-ce que Python ?

Python est un langage de programmation multi-usage.


Dans quel domaine Python se démarque-il ?

Python est utilisé pour plusieurs raisons, mais il se démarque en analyse de données et en machine learning (IA).


Comment puis-je commencer à programmer avec python ?

Il faut aller sur le site web officiel de python et télécharger le l'interpréteur du langage de programmation. Ensuite, il vous faudra un éditeur de texte et vous êtes prêts à débuter. Il ne vous reste qu'à apprendre.


Où puis-je trouver des tutoriels ?

On peut trouver une documentation sur le site web officiel de python. Pour ceux qui sont plus visuels, je vous conseille de faire un tour sur Youtube où plusieurs créateurs de contenu font plein de tutoriels pour bien débuter.


Python est-il un bon langage de programmation pour un débutant ?

Je suis un débutant moi-même et j'ai commencé avec ce langage. Je l'ai trouvé beaucoup plus simple à prendre en main comparativement à beaucoup d'autres qui utilisent plus de ponctuations et qui peuvent être mélangeants. Il est aussi très intuitif si vous êtes un minimum fluent en anglais, il sera très facile de comprendre ce que les fonctions de bases font.

Ce site web est supporté par les navigateurs : Chrome, Opéra, Safari, Edge et FireFox.
