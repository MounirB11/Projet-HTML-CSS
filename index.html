Accueil - Python, c'est quoi ?

Python est un langage de programmation interprété, multiparadigme et multiplateformes. Il favorise la programmation impérative structurée, fonctionnelle et orientée objet. Il est doté d'un typage dynamique fort, d'une gestion automatique de la mémoire par ramasse-miettes et d'un système de gestion d'exceptions ; il est ainsi similaire à Perl, Ruby, Scheme, Smalltalk et Tcl.

Principalement, Python est utilisé pour les IA, c'est à dire les intelligences artificielles ainsi que pour l'analyse de données. Il faut savoir est le langage de programmation le plus populaire en ce moment, mais il n'est clairement pas le meilleur pour toutes les situations. Par exemple, il est loin d'être le plus rapide pour les calculs. C serait beaucoup plus approprié pour de longs calculs car il est 5 à 10 fois plus rapide. Pour le web, Javascript est beaucoup plus approprié. Parcontre, python est bien aimé des utilisateurs car il est très intuitif.
